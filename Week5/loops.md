# JavaScript Loops

## 1. `while` Loop
рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃћрЃЮрЃарЃћрЃЉрЃљ рЃЏрЃљрЃюрЃљрЃЏ рЃАрЃљрЃюрЃљрЃЏ рЃарЃљрЃдрЃљрЃф рЃърЃўрЃарЃЮрЃЉрЃљ рЃљрЃарЃўрЃА рЃГрЃћрЃерЃЏрЃљрЃарЃўрЃбрЃў.

```javascript
let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}
```

### рЃЎрЃћрЃюрЃбрЃў рЃарЃўрЃфрЃ«рЃЋрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЉрЃћрЃГрЃЋрЃЊрЃЋрЃљ  `[l, r]` рЃўрЃюрЃбрЃћрЃарЃЋрЃљрЃџрЃерЃў using `while`
```javascript
let l = 3, r = 10;
while (l <= r) {
    if (l % 2 !== 0) {
        console.log(l);
    }
    l++;
}
```

## 2. `do...while` Loop
рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў `while`, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ рЃћрЃарЃЌрЃ«рЃћрЃџ рЃњрЃљрЃюрЃ«рЃЮрЃарЃфрЃўрЃћрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃљрЃАрЃерЃў рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃћрЃЉрЃў.

```javascript
let i = 1;
do {
    console.log(i);
    i++;
} while (i <= 5);
```

### рЃўрЃњрЃўрЃЋрЃћ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ рЃљрЃ«рЃљрЃџрЃў рЃфрЃўрЃЎрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ
```javascript
let l = 3, r = 10;
do {
    if (l % 2 !== 0) {
        console.log(l);
    }
    l++;
} while (l <= r);
```

## 3. `for` Loop
рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЎрЃЮрЃЏрЃърЃљрЃЦрЃбрЃБрЃарЃў рЃџрЃБрЃърЃў

```javascript
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
```

### 
```javascript
let l = 3, r = 10;
for (let i = l; i <= r; i++) {
    if (i % 2 !== 0) {
        console.log(i);
    }
}
```

---

РюЁ  `while` рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ рЃБрЃфрЃюрЃЮрЃЉрЃўрЃљ

РюЁ  `do...while` рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ 1 рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљ рЃњрЃАрЃБрЃарЃА

РюЁ  `for` рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃбрЃћрЃарЃљрЃфрЃўрЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃќрЃћ (рЃљрЃЋрЃ«рЃАрЃюрЃўрЃЌ рЃЏрЃЮрЃњрЃўрЃЋрЃљрЃюрЃћрЃўрЃЉрЃЌ) рЃљрЃю рЃўрЃфрЃў рЃџрЃБрЃърЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃў рЃЊрЃљ рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃў
